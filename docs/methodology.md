# Méthodologie

Cette étude ne suppose pas que les variables numérologiques ont une validité réelle.

Elles sont traitées comme des variables catégorielles dérivées des dates de match.

L’objectif est de tester empiriquement :

- leur association avec les résultats H/D/A ;
- leur stabilité dans le temps ;
- leur éventuelle utilité comme filtre secondaire ;
- leur valeur ajoutée par rapport aux cotes bookmaker.

Principe important :

Aucune variable d’affinité équipe/date ne doit être calculée avec des matchs futurs.
